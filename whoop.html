WHOOP Case Study

Project Overview

This project is a case study of the WHOOP app, focusing on its user interface and user experience design. The goal was to analyze the app's features, usability, and overall design aesthetics.

Project Details

The WHOOP app is designed to help users track their fitness and health metrics. It provides insights into sleep, recovery, and strain, allowing users to optimize their performance.

  • Role: UI/UX Designer
  • Tools Used: Figma, Adobe XD
  • Duration: 3 months

Research

Research involved user interviews, competitor analysis, and usability testing. Key findings included the need for a more intuitive navigation system and clearer visual hierarchy.
